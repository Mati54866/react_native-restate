# Real Estate App 🏠

A full-stack mobile real estate application built with React Native, featuring Google authentication, dynamic property listings, and user profiles.

## 📋 Overview

This React Native application provides a seamless real estate browsing experience with modern features like Google authentication, property search, and detailed property views. Built with Expo, Appwrite, and TypeScript for optimal performance and scalability.

## 🛠️ Tech Stack

- **Expo SDK 52** - React Native development platform
- **React Native** - Mobile UI framework
- **Appwrite** - Backend services (auth, database, storage)
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS / NativeWind** - Utility-first styling
- **Expo Router** - File-based navigation

## 🔋 Key Features

- ✅ **Google Authentication** - Secure user sign-in
- ✅ **Property Listings** - Browse properties with search & filter
- ✅ **Property Details** - Comprehensive property information
- ✅ **User Profiles** - Customizable user settings
- ✅ **Explore Page** - Browse all property types
- ✅ **Latest & Recommended** - Curated property suggestions
- ✅ **Centralized Data Fetching** - Efficient API calls
